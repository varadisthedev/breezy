React.js~
Node.js
Express
mongodb
Oauth

## design qualities~

a clean glassmorphism design
minimal clutter, maximum focus
easy to understand UI & natural UX

## features of Breezy

seperate, secured user profiles

lightweight on browser, with faster caching
transparency is the goal, from the UI to data privacy
